HTML (HyperText Markup Language)

           

HTML वेब पेजों की संरचना (structure) बनाने के लिए एक मूलभूत भाषा है। यह तत्वों (elements) और टैग्स (tags) का उपयोग करके कंटेंट को व्यवस्थित करती है।

           

Basic HTML Document Structure

            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page Title</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <nav>
            <a href="#home">Home</a>
            <a href="#about">About</a>
        </nav>
    </header>
    <main>
        <h1>Main Heading</h1>
        <p>This is a paragraph of text.</p>
        <img src="image.jpg" alt="A descriptive image">
        <ul>
            <li>List item 1</li>
            <li>List item 2</li>
        </ul>
    </main>
    <footer>
        <p>&copy; 2024 My Website</p>
    </footer>
</body>
</html>
           
           

Creating Advanced Forms

           

HTML फ़ॉर्म्स यूज़र से डेटा इकट्ठा करने का एक महत्वपूर्ण तरीका है। यहाँ एक विस्तृत फ़ॉर्म का उदाहरण है, जिसमें कई तरह के इनपुट फ़ील्ड्स (input fields) शामिल हैं:

            
<form action="/submit-form" method="POST">
    <fieldset>
        <legend>User Information</legend>
        
        <label for="name">Full Name:</label>
        <input type="text" id="name" name="user_name" required>
        
        <label for="email">Email Address:</label>
        <input type="email" id="email" name="user_email" required>

        <label for="password">Password:</label>
        <input type="password" id="password" name="user_password" minlength="8" required>

        <label for="dob">Date of Birth:</label>
        <input type="date" id="dob" name="user_dob">

        <p>Choose your favorite coding language:</p>
        <input type="radio" id="html_lang" name="fav_lang" value="HTML">
        <label for="html_lang">HTML</label>
        <input type="radio" id="css_lang" name="fav_lang" value="CSS">
        <label for="css_lang">CSS</label>
        <input type="radio" id="js_lang" name="fav_lang" value="JavaScript">
        <label for="js_lang">JavaScript</label>

        <label for="message">Your Message:</label>
        <textarea id="message" name="user_message" rows="4" cols="50"></textarea>

        <label for="country">Country:</label>
        <select id="country" name="user_country">
            <option value="india">India</option>
            <option value="usa">USA</option>
            <option value="uk">UK</option>
        </select>

        <label>
            <input type="checkbox" id="newsletter" name="newsletter_consent">
            Subscribe to newsletter
        </label>

        <button type="submit">Submit</button>
    </fieldset>
</form>
           
       
       
           

CSS (Cascading Style Sheets)

           

CSS वेब पेजों को सुंदर और आकर्षक बनाने के लिए स्टाइलिंग प्रदान करती है। यह रंग, फ़ॉन्ट, लेआउट और एनिमेशन को नियंत्रित करती है।

           

A Flexible Layout with Flexbox

           

Flexbox एक शक्तिशाली एक-आयामी लेआउट सिस्टम है जो आइटमों को एक पंक्ति (row) या स्तंभ (column) में संरेखित (align) करता है।

            
.flex-container {
    display: flex;
    flex-direction: row; /* row | column */
    flex-wrap: wrap; /* nowrap | wrap */
    justify-content: space-between; /* start | end | center | space-around */
    align-items: center; /* stretch | start | end | center | baseline */
    height: 200px;
    border: 2px solid #00ffff;
    padding: 10px;
}

.flex-item {
    background-color: rgba(255, 255, 255, 0.1);
    color: white;
    padding: 20px;
    margin: 5px;
    text-align: center;
    font-size: 1.2rem;
    flex-grow: 1;
}
           
           

A Powerful 2D Layout with CSS Grid

           

CSS Grid एक दो-आयामी लेआउट सिस्टम है जो जटिल वेब पेज लेआउट बनाने के लिए पंक्तियों (rows) और स्तंभों (columns) दोनों का उपयोग करता है।

            
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 equal columns */
    grid-template-rows: auto 1fr auto; /* Header, Main, Footer */
    gap: 20px;
    width: 100%;
    max-width: 900px;
    margin: 0 auto;
    border: 2px solid #ff416c;
    padding: 20px;
}

.grid-header {
    grid-column: 1 / span 3;
    background-color: rgba(255, 255, 255, 0.1);
    padding: 15px;
    text-align: center;
}

.grid-main {
    grid-column: 1 / span 2;
    background-color: rgba(255, 255, 255, 0.05);
    padding: 20px;
}

.grid-sidebar {
    grid-column: 3;
    background-color: rgba(255, 255, 255, 0.08);
    padding: 20px;
}

.grid-footer {
    grid-column: 1 / span 3;
    background-color: rgba(255, 255, 255, 0.1);
    padding: 15px;
    text-align: center;
}
           
           

Creative 3D Button Animation

           

CSS एनिमेशन का उपयोग करके आप एक साधारण बटन को एक आकर्षक 3D इफ़ेक्ट दे सकते हैं।

            
/* CSS for 3D Button */
.button-3d {
    display: inline-block;
    padding: 15px 30px;
    font-size: 1.2rem;
    font-weight: bold;
    color: white;
    background: linear-gradient(135deg, #00ffff, #00b3b3);
    border: none;
    border-radius: 10px;
    text-shadow: 1px 1px 2px rgba(0,0,0,0.4);
    box-shadow: 0 5px 0 #008080;
    cursor: pointer;
    position: relative;
    top: 0;
    transition: all 0.1s ease;
}

.button-3d:active {
    top: 5px;
    box-shadow: 0 0 0 #008080;
}
           
       
       
           

JavaScript

           

JavaScript is a powerful scripting language that makes webpages dynamic and interactive. It enables you to manipulate the content of an HTML page, handle user events (like clicks), validate forms, and communicate with web servers. It's the brain of your website.

            
<script>
    // A function to handle a click event
    document.getElementById("myButton").addEventListener("click", function() {
        alert("Button was clicked!");
    });

    // A function to change an element's text after a delay
    setTimeout(function() {
        document.getElementById("greeting").innerText = "Welcome to our website!";
    }, 2000); // 2000 milliseconds = 2 seconds

    // A function to add or remove a CSS class
    const element = document.getElementById("myElement");
    element.classList.add("highlight");
    // Or to toggle a class
    element.classList.toggle("hidden");

    // Form validation example
    const myForm = document.getElementById("myForm");
    myForm.addEventListener("submit", function(event) {
        const emailInput = document.getElementById("email");
        if (!emailInput.value.includes("@")) {
            event.preventDefault(); // Stop the form from submitting
            alert("Please enter a valid email address.");
        }
    });
</script>
           
           

JavaScript's core strength lies in its ability to interact with the **Document Object Model (DOM)**. The DOM is a tree-like representation of your HTML page, and JavaScript allows you to access and modify any element within it. This is what makes features like animated sliders, form validation, and real-time content updates possible.
