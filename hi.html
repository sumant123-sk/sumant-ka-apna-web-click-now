JavaScript

JavaScript is a powerful scripting language that makes webpages dynamic and interactive. It enables you to manipulate the content of an HTML page, handle user events (like clicks), validate forms, and communicate with web servers. It's the brain of your website.

Basic Event Handling and DOM Manipulation

Here's how you can use JavaScript to listen for user actions and change the page's content dynamically.

<script>
    // A function to handle a click event
    document.getElementById("myButton").addEventListener("click", function() {
        alert("Button was clicked!");
    });

    // A function to change an element's text after a delay
    setTimeout(function() {
        document.getElementById("greeting").innerText = "Welcome to our website!";
    }, 2000); // 2000 milliseconds = 2 seconds

    // A function to add or remove a CSS class
    const element = document.getElementById("myElement");
    element.classList.add("highlight");
    // Or to toggle a class
    element.classList.toggle("hidden");
</script>

Form Validation Example

JavaScript is commonly used to validate form data on the client side before it's sent to the server. This provides immediate feedback to the user and improves the overall experience.

<script>
    const myForm = document.getElementById("myForm");
    myForm.addEventListener("submit", function(event) {
        const emailInput = document.getElementById("email");
        if (!emailInput.value.includes("@")) {
            event.preventDefault(); // Stop the form from submitting
            alert("Please enter a valid email address.");
        }
    });
</script>

JavaScript's core strength lies in its ability to interact with the **Document Object Model (DOM)**. The DOM is a tree-like representation of your HTML page, and JavaScript allows you to access and modify any element within it. This is what makes features like animated sliders, form validation, and real-time content updates possible.
