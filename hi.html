HTML (HyperText Markup Language)

HTML is the core language for creating web pages. It defines the structure and content of a webpage using a system of elements and tags. Think of it as the skeleton of a website, providing the basic layout and content organization. Without HTML, a webpage would be just a raw text file.

Basic Structure and Elements

Every HTML document starts with a `` declaration, which defines the document type. The entire page content is enclosed within the `` tag. The `` section contains meta-information about the document, like the title, character set, and links to external stylesheets. The `` section contains the visible content of the webpage.

<head>
    <meta charset="UTF-8">

    <title>My First HTML Page</title>
</head>

    <h1>Hello World!</h1>
    <p>This is a paragraph.</p>

</html>
